# dict

有道词典命令行版，采用youdao网页查词引擎构建

Originated from: http://www.oschina.net/code/snippet_942897_27191

因为原作者不持续更新，为了保证自己的日常使用，该项目由自己维护

# Feature
- 更简洁的输入，没有多余的参数
- 零依赖，但是需要联网，不支持本地词库
- 支持中英和英中翻译
